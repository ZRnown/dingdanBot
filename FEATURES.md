# 新功能说明

## 1. 第三方设置功能

### 使用方法

向机器人发送命令 `/setshequ` 来设置要获取订单的第三方。

### 功能说明

- 机器人会显示所有可用的第三方列表
- 点击第三方名称可以切换选中/取消选中状态
- 可以选择多个第三方
- 选择"全部"选项可以获取所有第三方的订单
- 点击"完成设置"保存配置

### 设置逻辑

- **选中特定第三方**：只获取选中第三方的订单
- **选择"全部"**：获取所有第三方的订单（默认行为）
- **未选中任何第三方**：等同于选择"全部"

## 2. 时间范围调整

### 变更内容

- **之前**：只获取"今天"的订单
- **现在**：获取"最近2天"的订单

### 自动清理

- 超过2天的订单会自动从数据库中删除
- 清理操作在每次同步订单时自动执行

## 3. 订单获取逻辑

### 按第三方过滤

当设置了特定的第三方后：
- 只获取选中第三方的订单
- 使用 `ShequId` 参数过滤订单

### 获取全部

当选择"全部"或未选中任何第三方时：
- 获取所有第三方的订单
- 不使用 `ShequId` 参数

## 使用示例

1. **设置第三方**：
   ```
   /setshequ
   ```
   然后点击要选择的第三方，最后点击"完成设置"

2. **查看订单**：
   向机器人发送包含抖音链接的消息，机器人会自动查找并同步订单

3. **自动同步**：
   程序会自动：
   - 启动时同步最近2天的订单
   - 每5分钟检查一次新订单
   - 自动清理超过2天的订单
